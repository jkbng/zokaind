## Tech Stack

- Pure HTML / CSS / Vanilla JS — zero dependencies, zero build step
- Deployed via **Vercel** (static hosting)

## Deploy to Vercel

1. Push this repo to GitHub
2. Import the repo at [vercel.com/new](https://vercel.com/new)
3. Vercel auto-detects static site — click **Deploy**

No build command or output directory needed.
